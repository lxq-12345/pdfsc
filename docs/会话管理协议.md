# 会话备忘协议

## 目标

确保任何对话在意外中断时，最多丢失15分钟的内容。

---

## 触发词汇

```
手动检查点：  "检查点" / "保存检查点"
恢复会话：    "继续[项目]" / "恢复上一个session"
最终归档：    "归档" / "保存本次对话"
```

---

## 三层防护机制

**第1层：自动检查点（每15分钟）**
```
每15分钟自动保存，对用户透明，无需操作。
文件命名：CHECKPOINT_[YYYY-MM-DD]_[HH-MM]_AUTO.md
```

**第2层：手动检查点（随时可用）**
```
用户说"检查点"时立即保存。
适用于：重要修改前、网络不稳定时、完成重大功能后。
文件命名：CHECKPOINT_[YYYY-MM-DD]_[HH-MM]_MANUAL.md
```

**第3层：最终归档（会话结束）**
```
用户说"归档"时，合并所有检查点生成完整会话档案。
文件命名：SESSION_[YYYY-MM-DD]_[主题].md
存放位置：.claude/SESSIONS/
```

---

## 标准工作流程

```
启动会话
  ↓
正常对话（每15分钟自动保存）
  ↓
关键时刻可说"检查点"手动保存（可选）
  ↓
会话结束说"归档"生成完整档案
```

---

## 中断恢复

```
发生中断后：
  说"继续[项目]" → Claude加载最近检查点，从该点继续
  说"开始新session" → 忽略检查点，重新开始
```

---

## 文件存储位置

```
位置：
  mem/
```
