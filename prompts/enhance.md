# Enhancement Prompt - 智能增强提示词

## 目标
在忠实还原版的基础上，进行有据可依的补充和扩展，提升文档质量。

## 防幻觉三等级约束

### 等级1（允许自由增强）
- **类型**: 格式重组、表达优化、步骤补全
- **规则**: 自由增强，属于理解原文后的表达优化
- **示例**: 把简略的步骤补全、优化句子流畅性

### 等级2（谨慎，需有来源）
- **类型**: 跨文档引用、技术参数补充
- **规则**: 必须来自同批次其他PDF，必须标注来源
- **示例**: 从其他TaiShan-200文档引用相关配置参数

### 等级3（绝对禁止）⚠️
- **类型**: 原文和其他PDF都没有依据的内容
- **规则**: 任何情况下不能出现
- **示例**: 编造不存在的配置参数、不实的技术特性

## 工作流程

```
输入：
- 忠实还原版MD（事实锚点）
- 同批次的其他PDF信息（可选引用）
- 转换规范要求

处理：
1. 基于还原版进行增强
2. 严格遵守三等级约束
3. 等级2的补充必须标注来源
4. 自检：无等级3内容

输出：
- Frontmatter完整
- 增强内容有据可依
- 无幻觉编造
```
