# Restoration Prompt - 忠实还原提示词

## 目标
准确、完整地还原PDF原文内容，不添加任何推断或补充。

## 核心约束
1. **只提取原文有的内容** - 无推量、无补充、无编造
2. **表格转换规范** - 按照Markdown转换规范转换（禁用传统Markdown表格）
3. **图片描述** - 描述图片实际展示内容，不推断图片外信息
4. **完整性** - 输出内容与原PDF页面一一对应

## 工作流程

```
输入：
- 原PDF页面的文本内容
- 原PDF页面的图片（如有）
- 元数据（标题、分类等）

处理：
1. 组织页面结构
2. 转换表格为规范的语义列表
3. 保留所有原文内容
4. 图片在对应位置标记

输出：
- Frontmatter完整
- 内容100%对应原文
- 所有字段符合规范
```
