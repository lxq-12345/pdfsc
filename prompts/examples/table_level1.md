# Table Conversion - Level 1 Example (2D)

## 说明
第一层次表格转换示例：简单对应关系，2维数据结构

## 原Markdown表格（禁止格式）
```markdown
| 符号 | 含义 |
|------|------|
| ⚠️  | 警告 |
| 🔧 | 配置 |
| ✅ | 完成 |
```

## 转换后的语义列表格式（推荐）
```markdown
符号说明：

- ⚠️  代表警告信息
- 🔧 表示配置操作
- ✅ 表示完成状态
```

## 适用场景
简单的键值对应、术语解释、符号说明等，3行以内的小表格。

## 关键点
- 用自然语言的列表形式组织
- 清晰表达对应关系
- 易于RAG向量化理解
