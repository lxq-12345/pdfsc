# PDF智转器 程序清单（含说明）

## 1. 核心程序（src）

- `src/pdfsc.py`
  - CLI 主入口，负责任务调度（convert/convert-batch/verify/stats）

- `src/converter.py`
  - 两步转换核心实现（restore + enhance）
  - 管理 offline/fallback/mock 模式与低质量标记

- `src/model_adapter.py`
  - 统一模型适配层
  - 支持 `anthropic/openai/qwen/ollama/mock`

- `src/extractor.py`
  - PDF 文本提取与结构化输出

- `src/detector.py`
  - PDF 类型分析（文本/混排/扫描）

- `src/vision.py`
  - 图片提取与视觉相关处理辅助

- `src/image_handler.py`
  - 图片插入、命名与模式处理（disabled/description-only/extract）

- `src/metadata.py`
  - Frontmatter 元数据生成

- `src/validator.py`
  - Markdown 规则校验与幻觉检测

- `src/config.py`
  - 配置管理与模型能力注册表加载

- `src/logger.py`
  - 日志初始化与日志级别管理

## 2. 脚本（scripts）

- `scripts/install.sh`
  - 依赖安装脚本（WSL 优先 `.venv_wsl`）

- `scripts/run.sh`
  - 通用运行脚本（历史兼容）

- `scripts/run_d2.sh`
  - D2 专用一键脚本：`flow-test/convert/batch`

- `scripts/test.sh`
  - 测试执行脚本（现有实现）

- `scripts/validate.sh`
  - 输出验证脚本（现有实现）

- `scripts/snapshot_*.py|.sh`
  - 会话快照与归档辅助脚本

## 3. 配置（config）

- `config/default.yml`
  - 默认配置

- `config/config.example.yml`
  - 用户配置模板

- `config/model_capabilities.yml`
  - 模型能力注册表（视觉/中文能力等）

- `config/d2_flow.yml`
  - D2 流程联调默认配置

## 4. 测试（tests）

- `tests/test_stage_*.py`
  - 分阶段能力测试

- `tests/test_d2_mock_flow.py`
  - D2 mock 联调测试

- `tests/test_d2_batch_summary.py`
  - 批量报告统计字段测试

- `tests/test_cases/`
  - 瓶颈专项测试样例
