## 2 环境参数要求

在安装和使用服务器前，请确保满足以下环境条件：

### 2.1 温度要求

服务器的温度要求如下：

**工作温度**：5℃～40℃（41℉～104℉），温度变化率不超过最大20℃/小时或5℃/15分钟。

**ASHRAE等级**：符合国际标准Class A2/A3。

**存储温度说明**：
- 24小时内存储：-40℃～+65℃（-40°F～149°F）
- 3个月以内存储：-30℃～+60℃（-22℉～+140℉）
- 6个月以内存储：-15℃～+45℃（5℉～113℉）
- 1年以内存储：-10℃～+35℃（14℉～95℉）

> **⚠️ 重要**: 温度变化过快可能影响设备寿命，应避免剧烈温度波动。

### 2.2 湿度要求

服务器的湿度要求如下：

**工作湿度**：8%～90%无冷凝。

**最大湿度变化率**：20%/小时。

**存储湿度说明**：
- 96小时内存储：8%～95%（40℃）
- 3个月以内存储：8%～85%
- 6个月以内存储：8%～80%
- 1年以内存储：20%～75%

### 2.3 海拔高度要求

- **工作海拔高度**: ≤3050m（10000英尺）
- **按ASHRAE 2015标准**:
  - Class A1/A2：超过900m，每升高300m，工作温度降低1℃
  - Class A3：超过900m，每升高175m，工作温度降低1℃
  - Class A4：超过900m，每升高125m，工作温度降低1℃

### 2.4 其他环境条件

- **电源**: 交流供电，规格见第3章
- **接地**: 2280无独立接地端口，通过电源线接地线接地
- **防静电**: 安装前必须建立防静电环境
- **通风**: 确保设备周围有足够的空气流通
- **清洁**: 定期清理设备周围环境，防止灰尘进入

> **📝 说明**: 单个风扇失效可能影响节点性能或产生温度告警，更换后可恢复。建议定期检查风扇状态。
