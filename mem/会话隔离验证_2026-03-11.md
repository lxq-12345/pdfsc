## 一、问题场景

**场景**：两个Claude会话窗口同时运行

**原始问题**（v2.0）：
- 缓冲文件冲突：mem/.snapshot_buffer.md 中混合两个会话的内容
- 快照命名冲突：同分钟生成的快照会互相覆盖
- 归档混乱：无法区分快照属于哪个会话

---

## 二、解决方案（方案A - 会话隔离）

### 2.1 会话ID生成

每次会话启动时自动生成唯一ID：

```
SESSION_ID = YYYYMMDD_HHMM_XXXX

例如：
  会话A启动于 2026-03-11 12:32 → SESSION_ID: 20260311_1232_a7f3
  会话B启动于 2026-03-11 12:35 → SESSION_ID: 20260311_1235_c2e9

  即使同秒钟，各自拥有唯一ID
```

### 2.2 文件命名隔离

```
【缓冲文件】
  会话A：mem/.snapshot_buffer_20260311_1232_a7f3.md
  会话B：mem/.snapshot_buffer_20260311_1235_c2e9.md

  完全隔离，互不干扰 ✅

【快照文件】
  会话A的快照：快照-2026-03-11-1300-20260311_1232_a7f3.md
  会话B的快照：快照-2026-03-11-1300-20260311_1235_c2e9.md

  同分钟生成也不会覆盖 ✅

【备忘文件】
  备忘-2026-03-11.md （保持不变）
  → 归档时扫描所有快照，包括所有SESSION_ID的
  → 统一整理为当日备忘 ✅
```

## 三、工作流程示例

```
【会话A】                           【会话B】
启动 → SESSION_ID: sid_a            启动 → SESSION_ID: sid_b
   ↓                                    ↓
生成缓冲 .snapshot_buffer_a.md     生成缓冲 .snapshot_buffer_b.md
   ↓                                    ↓
用户A：快照                        用户B：快照
   ↓                                    ↓
读取.snapshot_buffer_a.md ✅       读取.snapshot_buffer_b.md ✅
生成快照-HHMM-a.md ✅              生成快照-HHMM-b.md ✅
清空.snapshot_buffer_a.md ✅       清空.snapshot_buffer_b.md ✅

【日终归档】
用户A或B：归档
   ↓
扫描mem/snapshot/中所有快照（含所有SESSION_ID的）
快照-2026-03-11-*-a.md
快照-2026-03-11-*-b.md
   ↓
统一整理 → 备忘-2026-03-11.md ✅
```

## 四、代码实现

### 4.1 Python脚本（snapshot_handler.py）

```python
def __init__(self, root_dir=None, session_id=None):
    # 生成或使用传入的SESSION_ID
    if session_id is None:
        now = datetime.now()
        short_uuid = uuid.uuid4().hex[:4]
        self.session_id = f"{now.strftime('%Y%m%d_%H%M')}_{short_uuid}"
    else:
        self.session_id = session_id

    # 隔离的缓冲文件（带SESSION_ID后缀）
    self.buffer_file = self.root_dir / "mem" / f".snapshot_buffer_{self.session_id}.md"
```

### 4.2 快照文件命名

```python
def generate_snapshot(self, conversation_text):
    now = datetime.now()
    snapshot_name = f"快照-{now.year}-{now.month}-{now.day}-{now.hour}{now.minute:02d}-{self.session_id}.md"
    snapshot_file = self.snapshot_dir / snapshot_name
```
